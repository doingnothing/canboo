网站使用说明

此网站共有两个页面，首页和用户阅读页面。首页内容包括电子书展示和注册功能（搜索功能没有实现）。阅读页面包括电子书的渲染。

只有用户注册了，才能进入阅读页面。当某个用户登录后，他可以点击某本书下面的“我要去啃”，然后就可以进入阅读页面。当进入阅读页面时，网页会对得到的图书内容进行渲染。渲染参数获取的规则如下：首先后台通过数据库查阅该用户是否阅读过本书，若之前读过，则应用之前读过的参数并从上次阅读的章节开始。若没有则判断用户是否阅读过同类主题的图书，若有，则应用阅读了的最后一本同类主题的图书的参数。若没有则判断用户此前是否阅读过图书，若有，则应用最近用户阅读的图书的参数。否则，使用默认参数。

网站使用到的工具：MySQL数据库，Node.js服务器。图书存储格式为JSON文件。
